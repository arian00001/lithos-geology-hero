# Lithos Geology Hero

A stunning full-screen dark-themed hero section for the geology brand **Lithos**.

## Features
- Cursor-following spotlight reveal effect
- Premium animations (Ken Burns zoom + staggered text reveal)
- Responsive design
- Built with React 18, TypeScript, Vite, Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```
